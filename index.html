INTRODUCCIÓN

(Anteriormente era conocido como JADE) es un propulsor de plantillas de Node.js. Con el podemos realizar códigos en HTML de una sintaxis mucho más sencilla, clara y directa, no solo a la hora de escribir sino también a la hora de leer y modificar.

CONCEPTO

Es un motor de plantillas influenciado por Haml capaz de facilitar y acelerar la creación de estas. Haml, está siendo implementado por: Unspace Interactive y otras tiendas Rails profesionales los usan exclusivamente para sus proyectos, valorando su enfoque en la limpieza, legibilidad y velocidad de producción.

¿POR QUÉ USARLO?

Imaginemos un momento que queremos realizar una aplicación web dónde solo se use HTML. ¿Por qué cargar una biblioteca PHP donde puede ser más complicado? Para eso tenemos Pug, la cual nos hace mas fácil todo.

PASOS

IMPORTANTE

Importante necesitas tener Node.js instalado, luego dentro te apareceran dos opciones, escoge la versión que se adapte al sistema de tu computador.
Para empezar a usar PUG debemos tener instalado GULP, sino lo tienes debes instalarlo abriendo el símbolo del sistema dirígete al menú de aplicaciones de Windows y escribe símbolo del sistema o cmd. También puedes presionar la tecla Win + R te aparecerá un buscador y digitas (cmd.exe o cmd)
Deberás crear una carpeta en cualquiera de los dos discos locales de tu pc, en la que vas a trabajar el proyecto
Dirígete al símbolo del sistema y accede a la carpeta que creaste anteriormente así: CD.. "esto para retroceder en carpetas y llegar a la deseada". Para entrar a la carpeta escribes CD (NOMBRE DE LA CARPETA). Para limpiar el cmd se escribe CLS
Una vez dentro de la carpeta creamos un archivo de inicio. El cual se crea con la siguiente Extensión: npm init. Llenamos los campos con la información requerida, luego de esto nos aparece la siguiente pregunta is this ok? damos enter para aceptar.
Esto creará automáticamente un archivo en la raíz de nuestro proyecto llamado PACKAGE.JSON.
Para usar PUG debemos instalar GULP. El cual se instala así npm install -- save - dev gulp Cuando finaliza la instalación de GULP en la raíz de nuestro proyecto se crea automáticamente la carpeta de módulos que es donde se encuentran los paquetes
Nos dirigimos a la raíz del proyecto para crear un archivo de texto que llamaremos GULPFILE.JS (se le cambia la extensión) en el cual especificaremos las tareas o tasks que queremos ejecutar.
Para convertir los archivos PUG a HTML se debe instalar PUG con la extensión: npm install -- save - dev gulp-pug
